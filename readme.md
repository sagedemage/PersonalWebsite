## About
My Personal website in React.

## Installation
Install these dependencies
```
npm install react react-dom react-router react-router-dom react-bootstrap bootstrap serve
```

## Try it out
Test web app
```
npm start
```

Build web app
```
npm run build
```

Serve the built web app for production
```
serve -s build
```
